# Admin Module Duties

The Admin Dashboard acts as the control tower for DealShaq.  
It focuses on governance, oversight, compliance, and reporting.

## Duties
- DRLP Management: approve/reject onboarding, suspend/remove non-compliant DRLPs
- Charity Management: maintain approved charity list, validate DRLP and DAC selections
- Contribution Oversight: aggregate Net Proceed contributions, execute disbursements, issue receipts
- Reporting & Analytics: dashboards, compliance reports, performance trends
- Audit & Compliance: monitor logs, validate DRLPDAC-List and DRLPDAC-SNL accuracy
- System Configuration: manage taxonomy, discount thresholds, DACSAI radius rules
- User Support & Overrides: handle escalations, corrections for errors
