# Backend Services Placeholder

The backend layer powers DealShaq by handling data integrity, aggregation, and system-wide services.

## Duties
- Net Proceed computation for both Retailer and Consumer transactions
- Monthly aggregation of contributions
- Generation of DRLPDAC-SNL (notification lists)
- Validation services (barcode, expiry, quantity, onboarding info, favorites)
- Audit logging and anomaly detection
- Disbursement file generation for Admin execution
- Centralized enforcement of discount rules and compliance policies
